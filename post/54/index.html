【Vue.js】Vue-cli的安装

博文 作者：广树 时间：2017-08-20 18:14 分类：JavaScript

1、首先要安装好Node.js


2、然后推荐将npm更换成cnpm： 

//更换npm的源地址
npm config set registry https://registry.npm.taobao.org  

// 配置后可通过下面方式来验证是否成功  
npm config get registry

//使用cnpm替代默认的npm
npm install -g cnpm --registry=https://registry.npm.taobao.org

然后就可以用cnpm来替换npm的操作了。

期间我发生了安装失败的情况，用管理员权限就好了。


3、开始安装Vue-cli

3.1、安装vue命令环境 

cnpm install vue-cli -g
3.2、验证安装 
vue --version
3.3、生成项目模板 
vue init <模板名> 本地文件夹名称

如 

vue init webpack webpack-demo
然后一路回车

3.4、进入项目目录 

cd 目录路径
cnpm install
3.5、运行 
npm run dev
